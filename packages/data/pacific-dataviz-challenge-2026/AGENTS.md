## Pacific Dataviz Challeng 2026 - data fetching and analysis

Everything in this directory is responsible for fetching and analyzing data which will be used for the Pacific Dataviz Challenge 2026.

### Instructions

* [README.md](./README.md) tracks the current progress of the data analysis including caveats, story spines, and citations. Record all meaningful data analysis discoveries to this file. If using web search to find additional data, please save that data to the `raw` subdirectory and **make sure it is referenced as a citation**.
* Use [KEY.md](./KEY.md) to understand the folder structure of this directory.
* You may incorporate authoritative data from external sources to augment this data analysis project. [The Pacific Data Hub](https://pacificdata.org/) is the preferred data source, but you may also look elsewhere.
 
### Other information

The actual implementation of the forthcoming dataviz project is located in the [packages/web/src/pages/pacific-data-viz-challenge-2026](../../web/src/pages/pacific-data-viz-challenge) directory. The README in that directory is meant to remain human authored, so avoid writing to that file unless requested.
